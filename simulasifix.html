📈 Eigenvalue (Eₙ)

Energi terkuantisasi: Eₙ = n² · (h²/8mL²)

Eₙ ∝ n² , n = 1,2,3...

📉 Eigenfunction (ψₙ)

ψₙ(x) = √(2/L) sin(nπx/L) , 0 < x < L

|ψ|² = probabilitas

🔍 Interpretasi: Node = titik simpul (ψ=0), jumlah node = n-1. Fungsi eigen ortogonal dan ternormalisasi.
